## Installation

1. Create and activate a virtual environment: 
Open a terminal, go the directory of the project, and type:
```sh
python3 -m venv .venv
source .venv/bin/activate
```

2. Install the required dependencies:
```sh
pip install -r requirements.txt
```

## Run the menu
```sh
python main.py
```
